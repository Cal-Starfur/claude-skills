## Mobile Bridge тАФ CRITICAL (Session 7, still live)

| Platform | Webview тЖТ Host | Host тЖТ Webview |
|---|---|---|
| Desktop | `window.parent.postMessage(obj, '*')` | Arrives as `{ type:'devvit-message', data:{ message:obj } }` |
| Mobile RN | `window.ReactNativeWebView.postMessage(JSON.stringify(msg))` | Same envelope, `e.origin === ""` |

```js
function postToHost(msg) {
  try {
    if (window.ReactNativeWebView) {
      window.ReactNativeWebView.postMessage(JSON.stringify(msg));
      return;
    }
    var target = (window.parent && window.parent !== window) ? window.parent : window;
    target.postMessage(msg, '*');
  } catch(e) {}
}

window.addEventListener('message', function(e) {
  // DO NOT gate on e.origin тАФ mobile RN delivers empty/null origin
  var msg = e.data;
  if (msg && msg.type === 'devvit-message' && msg.data && msg.data.message) {
    msg = typeof msg.data.message === 'string'
      ? JSON.parse(msg.data.message) : msg.data.message;
  }
  if (!msg || !msg.type) return;
  // ... handlers
});
```

## Coordinate System & Viewport

```
WORLD_W = 1194px  тАФ fixed logical world width (iPad Pro 11" landscape)
W = viewport width (varies per device, updated every resize)
H = viewport height (fills screen, updated every resize)

Desktop (W > WORLD_W):  centreOffsetX = (W-WORLD_W)/2, camX = 0
Mobile (~390px wide):   centreOffsetX = 0, camX range тЙИ 700px
```

**`ctx.translate(centreOffsetX - camX, 0)`** at draw() start тАФ combines centring + scroll.
**Pointer тЖТ world:** `mX = screenX - centreOffsetX + camX`
**Background elements** (sky, ground, sun, moon, stars, grass, flowers) use `WORLD_W` not `W`.
**`ctx.restore()`** before HUD тАФ karma, weather, death screen are screen-space.

### Snoo Cinematics тАФ CRITICAL (S18 lesson)
All `drawSnoo*` functions must be called **inside** `ctx.translate(centreOffsetX - camX, 0)`.
Never call them after `ctx.restore()`. World coords work natively тАФ no conversion needed.
Feed Snoo is the reference тАФ drain Snoo fixed in S18 by moving inside world transform.

## Drain System

### Down Drain
1. Worm digs tier 2 тЖТ sump (y=3H)
2. Holds still `JUNCTION_HOLD_FRAMES=90` (~1.5s), `_hmoved < 0.25`
3. Stamps `{sumpExit:true}`, pushes null (seals segment), sets `window._sumpHadDown=true`

### Up Drain
1. Requires `_sumpHadDown === true`
2. Returns to sump, holds still `_hmoved < 0.12`
3. Stamps `{sumpExit:true}` тАФ segment stays open
4. Digs upward тЖТ `head.y <= 2*H` fires +100 karma bonus

### тЪая╕П Naming Gotcha
`sumpExit: true` marks both drain types. Check `nextAfterExit`:
- `!nextAfterExit` тЖТ down-drain (tea flows to sump)
- `nextAfterExit` тЖТ sets `d.upDrain = true`

### Weekly Drain (Snoo Cinematic)
Fires in `updatePhysics()` when `nowW - weekStartTs >= WEEK_DRAIN_MS`.
`weekStartTs` is server-authoritative via `KV_WEEK`, synced on open + broadcast after each drain.
Cinematic phases: `floatin тЖТ pause тЖТ openvalve тЖТ draining тЖТ closevalve тЖТ floatout`

## Devvit Platform Rules (Hard-Won тАФ Never Violate)

| Rule | Why |
|---|---|
| PNG only for assets | GIF/JPG rejected by uploader |
| Push binaries via GitHub API directly | Sync script corrupts binary files |
| `webView.mount()` only in `onPress` | In render body = fires every render = crash |
| External `fetch()` silently fails | Devvit sandbox blocks non-Reddit HTTP |
| Channel names: `[a-zA-Z0-9_]` only | Colons crash Realtime channel render |
| `useChannel` declared after `useWebView` | Scope dependency |
| Always `git pull` before `devvit upload` | API push bypasses local git тАФ divergent branches break upload |
| Old posts go read-only after re-upload | Create new post to test every time |
| Devvit wraps `webView.postMessage()` in envelope | Must unwrap `{ type:'devvit-message', data:{ message } }` |
| Origin of hostтЖТwebview messages is NOT reddit.com | Remove strict origin check from game.js |
| All `drawSnoo*` calls inside `ctx.translate(centreOffsetX - camX, 0)` | After `ctx.restore()` = screen space = wrong on mobile |
| Background elements use `WORLD_W` not `W` | Sky, ground, sun, moon, stars, grass, flowers all use world width |
| `resizeCanvas()` always updates W/H | Never guard with `if (!W \|\| !H)` |
