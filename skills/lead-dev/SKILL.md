If it exists: read it before writing a single line.
If it doesn't: generate it immediately.

After every session:
- Update Function Registry with anything added/removed
- Add Changelog entry
- Flag new debt in Known Issues

## Hard Rules — Never Break These

1. Never start coding without reading the architecture
2. Never introduce naming that conflicts with existing conventions
3. Never duplicate a function that already exists
4. Never rename without a full find/replace audit
5. Never leave dead code
6. Never use magic numbers — always named constants
7. Never mix platform concerns (Devvit: blocks logic stays in main.tsx)
8. Never skip the version bump on output
9. Never end a session without updating GAME_ARCHITECTURE.md
10. Never take a shortcut without logging it as tech debt
11. After every push to GitHub: immediately check build status via the devvit-pipeline skill, report the result to the user, and if it passed — tell them to run `devvit upload` in their Codespace. Never wait for the user to ask.

---

## Devvit Rules (auto-applied when detected)

- All postMessage types = named constants, never raw strings
- No localStorage — use Redis via blocks
- Redis keys namespaced: `gameName:userId:keyName`
- Game logic in webview JS — never in main.tsx
- Reddit API calls in main.tsx — never in webview
- Every sent message type must have a case handler

## Bootstrap — All Scripts (Every Session)

Scripts live at `skills/lead-dev/scripts/` and `skills/lead-dev/tools/` in `Cal-Starfur/claude-skills`.

```bash
python3 << 'BOOTSTRAP'
import urllib.request, json, base64, sys
from pathlib import Path

TOKEN = sys.argv[1] if len(sys.argv) > 1 else input("Paste your GitHub PAT: ").strip()
headers = {
    "Authorization": f"token {TOKEN}",
    "Accept": "application/vnd.github.v3+json",
    "User-Agent": "LeadDev/1.0"
}

base = "https://api.github.com/repos/Cal-Starfur/claude-skills/contents/skills/lead-dev"
fetch = {
    "tools/parse.py":                   "tools/parse.py",
    "tools/session_log.py":             "tools/session_log.py",
    "tools/score.py":                   "tools/score.py",
    "tools/palette.py":                 "tools/palette.py",
    "scripts/audit.py":                 "scripts/audit.py",
    "scripts/generate_architecture.py": "scripts/generate_architecture.py",
    "scripts/drift_detect.py":          "scripts/drift_detect.py",
    "scripts/version_bump.py":          "scripts/version_bump.py",
    "scripts/review_code.py":           "scripts/review_code.py",
    "scripts/analyze_patterns.py":      "scripts/analyze_patterns.py",
    "scripts/propose_improvements.py":  "scripts/propose_improvements.py",
}

for local_path, remote_path in fetch.items():
    url = f"{base}/{remote_path}"
    req = urllib.request.Request(url, headers=headers)
    with urllib.request.urlopen(req) as r:
        data = json.loads(r.read())
        code = base64.b64decode(data["content"]).decode("utf-8")
    target = Path(f"/tmp/lead-dev/{local_path}")
    target.parent.mkdir(parents=True, exist_ok=True)
    target.write_text(code)
    print(f"✓ {local_path} ({len(code.splitlines())} lines)")

print("Bootstrap complete.")
BOOTSTRAP
```
